How to calculate Moon rise times

This is only an overview of the methodology. For detailed calculations, refer to the source code or to Chapter 47 of Astronomical Algorithms, Meeus 1998.

Given: day, month, year, latitude

  1. Find the Julian Date for that day, month, and year.
  2. Find the right ascension and declination of the Moon for that Julian Date. This will likely require the date to be nutated (finding the longitude and obliquity).
  3. Using the found declination, given latitude, and an altitude of 0, calculate the hour angle at Moon rise.
  4. Using the hour angle and right ascension, find the time of Moon rise in local sidereal time.
  5. Convert local sidereal time to local time.
